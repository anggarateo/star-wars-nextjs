## Requirement

Node >=20.16.0

NPM >=10.8.1

## Setup

Rename .env.local.example to .env.local

Install the dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```
